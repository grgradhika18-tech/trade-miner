🚀 Overview

This project aims to provide insights into international trade patterns by combining structured trade data with unstructured news data.
By leveraging APIs, it helps in understanding how global events, policies, and economic shifts impact import-export dynamics.

📊 Features
📈 Real-time trade data analysis
📰 News-based sentiment/context integration
🔄 Automated data fetching using APIs
📉 Visualization of import-export trends
📌 Correlation between global events and trade fluctuations


🛠️ Tech Stack
Language: Python
